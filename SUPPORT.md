# Support

Thanks for using Talaria.

## Where to ask for help

- Usage questions and troubleshooting: open a GitHub Discussion (if enabled) or GitHub Issue.
- Bug reports: open a GitHub Issue with repro steps, expected behavior, and logs.
- Security issues: please follow the private disclosure process in [SECURITY.md](SECURITY.md).

## What to include

Please include:

- Talaria version/commit SHA
- How you installed Talaria (`pip`, editable install, Docker)
- Runtime mode (single-repo or stable/dev)
- Relevant env vars (`TALARIA_BASE_URL`, `TALARIA_WORK_DIR`, `TALARIA_TELEGRAM_UI_ENABLED`)
- Exact command(s) run
- Error output and `logs/talaria.log` excerpt

## Response expectations

This is an OSS project and support is best-effort. Clear repros are usually resolved much faster.
